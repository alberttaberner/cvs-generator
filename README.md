# CVS Generator

## Description

Tool designed to provide random user data using the format "user"|"city"|"phone".
Data will be stored in the project's root path.

## Usage

```
python3 main.py
```

## Parameters

- Number of outputs
- Name gender: "all", "male", "female"
- File name + extension
